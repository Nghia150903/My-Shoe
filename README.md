# Website Thương mại điện tử My Shoe

## Giới thiệu

Đây là website thương mại điện tử bán giày được xây dựng bằng PHP và MySQL.

## Chức năng

- Đăng ký
- Đăng nhập
- Tìm kiếm sản phẩm
- Lọc sản phẩm
- Thêm vào giỏ hàng
- Thanh toán

## Công nghệ sử dụng

- PHP
- MySQL
- HTML
- CSS
- JavaScript
- Bootstrap
- XAMPP

## Hướng dẫn cài đặt

1. Clone repository

```bash
git clone https://github.com/username/my-shoe.git
```

2. Chép project vào thư mục `xampp/htdocs`.

3. Import file `database/shoe.sql` vào MySQL.

4. Khởi động Apache và MySQL trong XAMPP.

5. Truy cập:

```
http://localhost/shoe_php
```
